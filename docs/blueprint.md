# **App Name**: Ramadan Rewards Quiz

## Core Features:

- Quiz Questions: Display Ramadan-related multiple-choice questions on the first page.
- Answer Submission: Allow users to select and submit their answers to each question.
- Score Calculation: Calculate the percentage of correct answers in real-time.
- MB Reward Display: Show the equivalent MB reward based on the score on the second page.
- Try Again Functionality: Provide a 'Try Again' button to navigate back to the quiz and retake it.
- Phone Number Input: Allow users to enter their phone number to claim their MB reward on the third page.
- Reward Confirmation: Display a congratulatory message and confirm that the MB will be sent within 3-5 hours on the fourth page.

## Style Guidelines:

- Primary color: Deep purple (#4A148C) to represent spirituality and the night sky during Ramadan.
- Background color: Light lavender (#E1BEE7), a desaturated version of the primary, to provide a calming backdrop.
- Accent color: Gold (#FFD700) for highlights and CTAs, symbolizing celebration and rewards.
- Body and headline font: 'Poppins' for a contemporary, precise feel.
- Use icons representing Ramadan traditions such as mosques, dates, crescents, and lanterns, with a modern, animated style.
- Employ a clean, user-friendly layout that is optimized for mobile devices.
- Incorporate subtle animations to transition between pages, provide feedback on quiz answers, and celebrate successful submissions.
